# LumenUI: Modern Animated Component Library for Web Apps

LumenUI is a fast, modern, developer friendly component library built for designers, developers, and product teams who want beautifully crafted, production ready UI components with minimal setup. It provides a clean collection of responsive, customizable, and animation-enhanced components powered by Tailwind CSS, Shadcn UI, and Framer Motion.

Each component comes with a live preview, instant updates, motion effects, installation commands, and ready-to-copy source code helping you build polished interfaces without repetitive styling or boilerplate.
The entire library is intuitive, lightweight, and fully optimized for Next.js, offering smooth performance, quick navigation, and a delightful developer experience.

---

## Tech Stack

**Frontend**

- Next.js

- Tailwind CSS

- Framer Motion 

- Shadcn UI / Custom Components

- Lucide Icons

- TypeScript

**Development**

- Next.js App Router

- Modular component architecture

- TypeScript

## Setup & Installation

Follow these steps to run FluxInvoice locally:

1. **Clone the Repository:**
```bash
git clone https://github.com/Prathik018/lumenui.git
cd lumenui
```

2. **Install Node Module:**
```bash
npm i
```

3. **Start the Development Server:**
```bash
npm run dev
```
